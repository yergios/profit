# ProfIt

ProfIt is a single-page web app that helps small entrepreneurs — bakers,
freelancers, Airbnb hosts, creatives — calculate the true cost of their
products or services and determine a profitable selling price. No login,
no backend, no data stored. Works offline. You land on the page and you're ready to calculate.

## How to run locally

1. Open `src/index.html` in your browser
